# Aprendizado-Rust

Códigos desenvolvidos ao aprender a Linguagem de Programação Rust

Esse repositório faz uso da ferramenta [mask](https://github.com/jacobdeichert/mask)

## run
> Roda o arquivo main, usando o arquivo de input e retorna para o output
```sh
cargo run -q - < IO/input.txt > IO/output.txt
clear
cat IO/output.txt
```

## teste
> Compara a saída do programa com o exemplo
```sh
diff IO/output.txt IO/example.txt
```
